### Week 2 Objectives:

* Learn how to build network infrastructure on AWS, deploy EC2 instances, and configure networking components to create a secure and reliable environment.

| Day | Tasks | Start Date | End Date | References |

### Week 2 Achievements:

* Gained a solid understanding of AWS networking through **Amazon VPC**.

* Successfully created VPCs, Subnets, Internet Gateways, and Route Tables to build private cloud networks on AWS.

* Configured Security Groups and Network ACLs to control inbound and outbound network traffic.

* Enabled VPC Flow Logs to monitor and analyze network traffic.

* Successfully deployed EC2 instances and verified Internet connectivity.

* Became familiar with NAT Gateway, Reachability Analyzer, and Session Manager for instance management.

* Used Amazon CloudWatch to monitor EC2 instance performance and health.

* Acquired fundamental knowledge of **Infrastructure as Code (IaC)** for automating infrastructure deployment.
