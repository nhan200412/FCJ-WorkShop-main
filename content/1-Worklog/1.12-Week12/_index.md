### Week 12 Objectives:

* Participate in developing and presenting a serverless backend project on AWS while applying cloud architecture concepts to build a scalable, cost-efficient, and highly available system.

### Week 12 Achievements:

* Gained a solid understanding of AWS Serverless architecture and its core components.

* Applied Amazon API Gateway to receive and route client requests.

* Implemented business logic using AWS Lambda with automatic scaling capabilities.

* Used Amazon DynamoDB as a high-performance NoSQL database for order processing.

* Configured AWS KMS to encrypt customer data and enhance security.

* Understood how serverless architecture reduces infrastructure management and operational costs.

* Learned about Lambda Cold Start issues and the use of Provisioned Concurrency to improve performance.

* Successfully presented the team project and strengthened practical AWS cloud architecture skills.
